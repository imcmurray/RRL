# Rinse Repeat Labs — Company Context

## About Us

Rinse Repeat Labs is a software development agency specializing in building mobile, web, and desktop applications. We partner with clients to bring their ideas to life through high-quality, well-engineered software.

**Website:** https://rinserepeatlabs.com

## Our Services

### Mobile Development
- Native iOS (Swift/SwiftUI)
- Native Android (Kotlin/Jetpack Compose)
- Cross-platform (Flutter, React Native)

### Web Development
- Frontend (React, Vue, Next.js)
- Backend (Node.js, Python, Go)
- Full-stack applications

### Desktop Development
- Electron applications
- Native macOS/Windows apps

### Additional Services
- UI/UX design
- Technical consulting
- MVP development
- App maintenance and support

## Our Team

We operate with a lean, senior team augmented by AI-powered agents for planning, coordination, and decision-making:

- **CITO** — Chief Information Technology Officer (technical strategy)
- **PM** — Project Manager (delivery coordination)
- **DevLead** — Lead Developer (architecture, implementation)
- **QALead** — QA Lead (quality assurance)
- **CustomerSuccess** — Customer Success Manager (client relationships)
- **Marketing** — Marketing/Growth Lead (launch strategy, ASO)

## Our Values

1. **Quality over quantity** — We'd rather do fewer projects excellently than many projects poorly
2. **Clear communication** — No surprises for clients or team members
3. **Pragmatic engineering** — Right-sized solutions, not over-engineered
4. **Continuous improvement** — Learn from every project
5. **Client partnership** — We succeed when our clients succeed

## Working Model

- **Sprint-based delivery** — 2-week sprints with clear deliverables
- **Regular communication** — Weekly client updates, daily internal standups
- **Transparent progress** — Clients have visibility into project status
- **Quality gates** — Code review, testing, and QA sign-off before releases

## Technology Preferences

### Mobile
- Flutter (preferred for cross-platform)
- Swift/SwiftUI (iOS-only projects)
- Kotlin (Android-only projects)

### Web
- React/Next.js (frontend)
- Node.js or Python (backend)
- PostgreSQL (database)

### Infrastructure
- AWS or GCP (cloud)
- GitHub (version control)
- GitHub Actions (CI/CD)

## Client Engagement Model

1. **Discovery** — Understand requirements, define scope
2. **Proposal** — Technical approach, timeline, pricing
3. **Development** — Sprint-based delivery with regular demos
4. **Launch** — QA sign-off, deployment, ASO support
5. **Support** — Ongoing maintenance and enhancements
